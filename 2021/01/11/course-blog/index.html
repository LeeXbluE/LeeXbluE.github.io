HEXO 博客搭建历程

本人用到的网站

  1. hexo 官方文档
  2. valine–无后端评论系统
  3. Butterfly主题文档
  4. GitHub Pages–用于部署静态网页

使用的小插件

  1. 豆瓣_hexo-douban由于豆瓣开启了反盗链，因此，一些浏览器（safari）会无法显示图片，可把meta设为true。但此操作会导致一些插件失效（例如：不蒜子）。
  2. 本地搜索_hexo-generator-search
  3. hexo-wordcount

注意点

  1. 使用了butterfly的主题，使用了npm安装方式，所以在themes文件夹中没有生成butterfly的文件夹，开始时略困惑，后来做测试的时候使用git安装是会生成的。现在还不知道这个文件夹是否有用。
  2. 一直想吧主题封面换掉，跟着文档，以https 格式获取到的图片是可以显示的，但本地图片却行不通，开始纠结绝对路径、相对路径之类，纠结相对路径中的,/,//,\ 四种有什么区别(因为有些地方格式要求严格到吐血，下次再看这个)。后来看了hexo文档中对资源文件夹的叙述，才知道hexo的资源根目录是source，就此解决。
  3. Pjax功能，文档说容易出现问题。

待解决

  1. 对github上的秘钥之类的事不够了解
  2. github中 private的资源与public的资源使用区别
  3. github 分支用法用处 ，master分支等
  4. meta是什么？
  5. CDN是什么？
  6. 后续可实现PWA
  7. hexo clean && hexo g && hexo d 三件套
  8. repo
    LeanCloud 的相关准备
